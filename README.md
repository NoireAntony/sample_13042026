## 🚀 О проекте

Данный репозиторий создан для демонстрации базового рабочего процесса с системой контроля версий **Git**. Здесь вы найдёте справочную информацию по основным командам, необходимым для повседневной работы.

---

## 💻 Основные команды Git

### 1️⃣ Добавить изменения в индекс

```bash
git add .
```

> 💡 Для добавления конкретных файлов используйте: `git add <файл1> <файл2>`

### 2️⃣ Создать коммит

```bash
git commit -m "Описание коммита"
```

> ✏️ Пишите понятные и кратные описания изменений.

### 3️⃣ Отправить в удалённый репозиторий

```bash
git push origin <имя-ветки>
```

**Примеры:**
```bash
git push origin main
git push origin master
```

---

## 🔗 Первая публикация в репозитории

Если это ваш **первый коммит** и ветка ещё не связана с удалённым репозиторием, используйте:

```bash
git push -u origin <имя-ветки>
```

> ✅ Эта команда привяжет локальную ветку к удалённой. В дальнейшем достаточно будет просто `git push`.
